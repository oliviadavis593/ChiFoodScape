🥦 ChiFoodScape

An open-source map of real food access in Chicago.
Built using city inspection data, Python, and spatial analysis.

Project Goals

  • Reveal true grocery access beyond city records
  • Distinguish real grocery stores from junk/low-nutrition outlets
  • Support food justice and community planning
